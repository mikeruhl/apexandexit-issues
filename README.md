## Guidelines

- **Search first**: Check if your issue or feature request already exists before creating a new one
- **Be specific**: Provide clear descriptions and reproduction steps for bugs
- **One issue per report**: Don't combine multiple bugs or features in a single issue
- **Be respectful**: Follow our code of conduct and treat others with respect

## Issue Labels

- `bug`: Something isn't working correctly
- `enhancement`: New features or improvements
- `duplicate`: This issue already exists
- `wontfix`: This will not be addressed
- `help wanted`: Community contributions welcome
